# 0.1.2 — 2026-05-31

## Removed

- **`osv_query_batch` DataCanvas spillover** — `canvas_id` input field, `canvas_id` output field, `CANVAS_SPILL_THRESHOLD` constant, `getCanvas` import, and the spillover block removed from the handler and `format()`. No `dataframe_query`/`dataframe_describe` tool was ever added to consume the canvas handle, so the output was dead ([#1](https://github.com/cyanheads/osv-advisory-mcp-server/issues/1)).
- **`src/services/canvas/canvas-accessor.ts`** — `setCanvas`/`getCanvas` module deleted; `setCanvas(core.canvas)` call and `setup(core)` parameter removed from `src/index.ts`.
- **`devcheck.config.json` outdated allowlist** — `@cyanheads/mcp-ts-core` added to `outdated.allowlist` so the outdated gate stays green while framework adoption is deferred (framework published `0.9.19` mid-cycle).
